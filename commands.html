Commands¶

The Makefile contains the central entry points for common tasks related to this project.

Install dependencies¶

  • make requirements

Process raw dataset¶

  • make data

Run hyperparameter opt¶

  • make run

Flake8 Lint¶

  • make lint

Black Format¶

  • make format

Build sphinx documentation¶

  • cd docs && make html
